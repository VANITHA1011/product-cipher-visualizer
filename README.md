# 🔐 Product Cipher Visualizer

An interactive web-based cryptography tool that demonstrates **Caesar Cipher**, **Rail Fence Cipher**, and their **combined Product Cipher** with real-time visualization and analysis.

---

## 📌 Project Overview

This project helps users **learn cryptography visually** by showing how plaintext transforms step-by-step into ciphertext using substitution and transposition techniques.

---

## 🧠 Core Features

* 🔍 Step-by-step encryption visualization
* 🔐 Caesar Cipher (Substitution)
* 🔀 Rail Fence Cipher (Transposition)
* 🔗 Product Cipher (Combined Encryption)
* 📊 Strength Analysis (Key space comparison)
* 💥 Avalanche Effect demonstration
* 🎛️ Interactive controls for keys and parameters
* 📘 Educational explanations

## 🏗️ Architecture Explanation

1. User provides plaintext and parameters
2. Web interface captures input
3. Encryption controller manages process
4. Substitution (Caesar Cipher) applied
5. Transposition (Rail Fence) applied
6. Multiple rounds executed
7. Visualization engine shows transformation
8. Final ciphertext generated
9. Analysis module evaluates strength
